# Тест чек-листов в Markdown

## Простой чек-лист

- [x] Завершенная задача
- [ ] Незавершенная задача  
- [ ] Еще одна задача с **форматированием** и *курсивом*
- [x] Другая завершенная задача

## Чек-лист разработки

- [x] Создать дизайн
- [x] Написать код
- [ ] Протестировать функциональность
- [ ] Развернуть на продакшн
- [ ] Собрать обратную связь

## Вложенные чек-листы

- [x] Основная задача 1
  - [x] Подзадача 1.1
  - [ ] Подзадача 1.2
- [ ] Основная задача 2
  - [ ] Подзадача 2.1
  - [ ] Подзадача 2.2

## Смешанный список

- [x] Задача с чекбоксом
- Обычный элемент списка
- [ ] Еще одна задача
- Еще один обычный элемент

## В блокквоте

> ### Список задач для релиза:
> 
> - [x] Обновить документацию
> - [ ] Создать changelog  
> - [ ] Подготовить релизные заметки
> - [x] Протестировать на разных браузерах

## Обычный список (для сравнения)

- Первый элемент
- Второй элемент
- Третий элемент
  - Вложенный элемент
  - Еще один вложенный
