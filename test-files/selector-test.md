# Тест интерактивных селекторов

## Overview

**Revision:** v1.0  
**Status:** Active  
**Image:** ./assets/CLN17V3.0-Spin.gif

Тестовая страница для проверки работы интерактивных селекторов выбора.

**Key Features:**
- Синхронизация между селекторами
- Плавные анимации переключения
- Адаптивный дизайн
- Поддержка якорей

## Дополнительная информация

> **Note:** Селекторы с одинаковым набором опций автоматически синхронизируются. При выборе опции в одном селекторе, все селекторы ниже с такими же опциями обновятся автоматически.

> **Tip:** Вы можете использовать якоря для прямых ссылок на конкретные разделы. Например: `#selector-0#npm` приведет к первому селектору с выбранной опцией NPM.
