## Quick Start

```bash
npx create-beta-app my-app

npm run dev
```

## Architecture

Project Beta follows a modular architecture with these core components:

- **Router**: File-based routing with dynamic imports
- **Renderer**: Universal rendering engine
- **Optimizer**: Automatic performance optimization
- **Cache**: Intelligent caching layer

## Deployment

```bash
# Deploy to Beta Cloud
beta deploy

# Deploy to custom server
beta build
beta deploy --target custom --host example.com
```
