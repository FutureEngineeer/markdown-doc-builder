**Revision:** v1.0

**Tags:** Test, Selectors, Nested Headers

---

## Селектор с вложенными заголовками

Choose your framework: [ **React** ](#react) | [ **Vue** ](#vue)

### React

React — популярная библиотека для создания UI.

#### Установка

```bash
npm install react react-dom
```

#### Основные концепции

- Компоненты
- Props
- State
- Hooks

##### Hooks

Hooks позволяют использовать состояние в функциональных компонентах:

- useState
- useEffect
- useContext

#### Экосистема

React имеет огромную экосистему:

- React Router
- Redux
- Next.js

### Vue

Vue — прогрессивный фреймворк.

#### Установка

```bash
npm install vue
```

#### Основные концепции

- Реактивность
- Директивы
- Computed свойства

##### Composition API

Новый способ организации логики:

- ref
- reactive
- computed

#### Экосистема

Vue также имеет богатую экосистему:

- Vue Router
- Pinia
- Nuxt.js
