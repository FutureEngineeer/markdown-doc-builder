# Welcome to Documentation Site

## Project Overview

This is the home page of the documentation site. It demonstrates the new **root folder** functionality.

### Key Features

- **Simple Structure**: Files in root folder are placed at the site root
- **Automatic Home Page**: `home.md` becomes `index.html`
- **Section Support**: Folders inside root become sections
- **Flexible Configuration**: Use `doc-config.yaml` for custom structure

## Getting Started

1. Create a `home.md` file in your root folder
2. Add other `.md` files for additional pages
3. Create folders for sections
4. Optionally add `doc-config.yaml` in folders for custom structure
