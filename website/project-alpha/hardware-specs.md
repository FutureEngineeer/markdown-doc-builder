# Hardware Specs

## Processor

High-performance ARM Cortex-M4 microcontroller with floating-point unit.

### Core Features
- 32-bit ARM Cortex-M4 at 168MHz
- 1MB Flash memory
- 192KB SRAM
- Hardware floating-point unit (FPU)

### Performance Metrics
- 210 DMIPS processing power
- DSP instructions for signal processing
- Single-cycle multiply and accumulate

## Memory

Advanced memory architecture with multiple storage options.

### Internal Memory
- 1MB Flash for program storage
- 192KB SRAM for data processing
- 4KB backup SRAM with battery support

### External Storage
- MicroSD card slot (up to 32GB)
- SPI Flash interface for additional storage
- I2C EEPROM for configuration data

## Connectivity

Multiple communication interfaces for versatile connectivity options.

### Wired Interfaces
- USB 2.0 Full Speed (12 Mbps)
- Ethernet 10/100 Mbps
- RS485/RS232 serial communication
- CAN-FD bus support

### Wireless Options
- Wi-Fi 802.11 b/g/n (2.4GHz)
- Bluetooth 5.0 Low Energy
- LoRaWAN for long-range communication
