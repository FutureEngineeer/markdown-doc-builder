# Документация

Полная документация тестового сайта.

## Установка

```bash
node build-all-v3.js test-site
```

## Использование

1. Создайте структуру файлов
2. Настройте `config.yaml`
3. Определите иерархию в `doc-config.yaml`
4. Запустите сборку

## Конфигурация

### config.yaml

Основные настройки сайта:
- Название и описание
- Навигация
- Социальные ссылки
- Иконки

### doc-config.yaml

Структура документации:
- Иерархия страниц
- Порядок в меню
- Игнорируемые файлы
