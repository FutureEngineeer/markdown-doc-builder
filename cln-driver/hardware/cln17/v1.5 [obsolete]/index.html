CLN17 V1.5 HARDWARE

Compact driver for NEMA17, a corrected and improved version of CLN17 V1.0, suitable for 12V and 24V systems with a current of up to 1.4A RMS.

Tested hardware featuresüîó

Driver IC:

  • ‚úÖ TMC2209 STEP-DIR, UART SW bidirectional communication, Reset + Disable functionality
  • ‚úÖ Freewheeling when unpowered
  • ‚úÖ Analog Current sensing

Encoder IC:

  • ‚úÖ TLE5012 SPI communication
  • ‚úÖ TLE5012 through the PCB position sensing (1LSB noise)

Power:

  • ‚úÖ Daisy Chain Power connection
  • ‚úÖ DCDC stability at voltage range [External: 5.2V-25V, USB: 4.9V-5.5V]
  • ‚úÖ ENB CLTR 5V (Power Sink & Source 5V 200mA)

Interfaces:

  • ‚úÖ USB communication
  • ‚úÖ CTRL connector - UART, I2C, SPI communication (ABZ not tested yet)
  • ‚úÖ CAN-Bus IC Shutdown and Logic supply, diff signal generation
  • ‚úÖ Expansion connector - UART communication
  • ‚úÖ SWD with reset functionality

Peripherals:

  • ‚úÖ SW1 & SW2 as button
  • ‚úÖ RESET using SW2 as output
  • ‚úÖ RGB LED (single color indication)
  • ‚úÖ External Precise Clock (10ppm) (HSE)

  • PCB Surface Finish: ENIG [preferred]
  • Outer/Inner Copper Weight: 1oz/0.5oz
  • Via Covering: Plugged [preferred] or Tented
  • Board Outline Tolerance: ¬±0.2mm
