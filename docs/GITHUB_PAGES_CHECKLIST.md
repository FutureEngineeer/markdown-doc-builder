# ✅ Чеклист настройки GitHub Pages

## Перед деплоем

- [ ] Установлены все зависимости: `npm install`
- [ ] Обновлен `baseUrl` в `config.yaml`
- [ ] Локальная сборка работает: `npm run build:all`
- [ ] Проверен индекс: `npm run index:check`
- [ ] Проверена поиск: `npm run search:test`

## Настройка GitHub

- [ ] GitHub Pages включен в Settings → Pages
- [ ] Выбран источник: **GitHub Actions**
- [ ] Workflow файлы находятся в `.github/workflows/`
- [ ] Файлы закоммичены и запушены

## Проверка деплоя

- [ ] Workflow запустился в разделе Actions
- [ ] Все шаги выполнены успешно (зеленые галочки)
- [ ] Сайт доступен по адресу: `https://username.github.io/repo`
- [ ] Навигация работает
- [ ] Поиск работает
- [ ] Изображения загружаются

## Оптимизация

- [ ] Установлены пакеты оптимизации изображений
- [ ] Изображения оптимизированы: `npm run optimize:images:smart`
- [ ] Кеш работает корректно
- [ ] Размер dist/ приемлемый

## Автоматизация

- [ ] Деплой при пуше в main работает
- [ ] Автообновление репозиториев настроено (каждые 12 часов)
- [ ] Кеширование работает в GitHub Actions

## Мониторинг

- [ ] Логи сборки доступны в Actions
- [ ] Время сборки приемлемое (30-60 сек с кешем)
- [ ] Нет ошибок в консоли браузера

## Документация

- [ ] README.md обновлен с ссылкой на сайт
- [ ] Инструкции по настройке доступны
- [ ] Команды документированы

---

## Быстрые команды

```bash
# Локальная сборка
npm run build:all

# Оптимизация изображений
npm run optimize:images:smart

# Очистка кеша
npm run cache:clear

# Информация о кеше
npm run cache:info

# Проверка индекса
npm run index:check
```
